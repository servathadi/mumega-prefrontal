# Homepage Rulebook v1 — Prefrontal Club

## هدف
هوم‌پیج باید هر روز «تصویر زنده کسب‌وکار» را نشان دهد: رویدادها، دوره‌ها، کتاب‌های رایگان.

## ساختار پیشنهادی مرکزی (Middle Blocks)
1. **رویدادهای داغ (Featured Events)**
   - منبع: Event CPT
   - تعداد: 3
   - CTA: ثبت‌نام (Eventbrite/صفحه رویداد)

2. **دوره‌های ویژه (Featured Courses)**
   - منبع: LearnPress/Woo
   - تعداد: 3
   - CTA: مشاهده دوره / خرید

3. **کتابخانه رایگان (Free Books)**
   - منبع: محصولات 0 CAD + صفحه free-library
   - تعداد: 3
   - CTA: دریافت نسخه کامل رایگان

## قانون نام‌گذاری Featured
- Featured Event: `[FEAT-EVENT] <title>`
- Featured Course: `[FEAT-COURSE] <title>`
- Featured Book: `[FEAT-BOOK] <title>`

## قانون انتخاب آیتم برای هوم‌پیج
- رویداد: نزدیک‌ترین تاریخ + بالاترین اولویت ثبت‌نام
- دوره: فعال و قابل خرید
- کتاب: دانلود سالم + کاور مناسب

## حداقل محتوای هر کارت
- عنوان کوتاه
- یک خط ارزش پیشنهادی
- زمان/قیمت (اگر دارد)
- CTA واضح

## SLA بروزرسانی
- هر پست مهم کانال: حداکثر 6 ساعت تا انعکاس روی سایت
- هر آیتم Featured: بازبینی هفتگی

## تیم و نقش‌ها
- Prefrontal: تصمیم و چینش Featured
- Content Agent: متن کارت‌ها
- MCP Operator: اجرا روی WP
- Admin: تایید نهایی و کنترل کیفیت

## QA قبل از انتشار
- لینک CTA سالم
- تصویر شاخص وجود دارد
- موبایل قابل‌خواندن
- آیتم تکراری نیست
- با برند (فونت/رنگ/لحن) هماهنگ است

## KPI هوم‌پیج
- CTR روی CTA هر بلوک
- نرخ تبدیل بازدید به checkout
- نرخ دانلود کتاب رایگان
- نرخ ثبت‌نام رویداد
